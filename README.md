# Velvet & Vista - Multi-Destination Guest Journery Interface

## Overview
Velvet & Vista is a luxury interactive guest journery interface for "bipolar weddings," where events occur across two different destinations. Guests can visualize itinerary, check local times, see weather and track journey Progress.

## Features
- Dual Destination Timeline: Portugal and Morocco. Each has its own luxury themes.
- Mode Toggle: Switch between 'Logistics' and 'Celebration' events.
- Interactive Progress Tracker: Move Silder to simulate journey progress.
- Live Context Cards: Shows local time and weather for each event.
- Responsive Mobile-First Design: Fully Functional at widths as narrow as 360px
- Luxury UI: Serif typography, smooth transitions, high-quality imagery.

## Setup
### 1. Clone the Repository
git clone https://github.com/saqibaltaf27/Velvet-Vista.git

### 2. cd Velvet-Vista
### 3. Configure .env
VITE_OPENWEATHER_API_KEY=(Enter your own API Key)
### 4. Installing Dependencies
npm install
### 5. Run server
npm run dev
